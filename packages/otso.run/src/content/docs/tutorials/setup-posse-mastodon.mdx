## Connect account

Run:

```bash
otso login mastodon https://mastodon.social
```

Follow the OAuth flow and store the token.

## Choose kinds

Edit `otsorc.yml`:

```yaml
posse:
  mastodon:
    kinds: [note, photo]
```

## Test a post

Publish a note and confirm the Mastodon copy links back to your site.
