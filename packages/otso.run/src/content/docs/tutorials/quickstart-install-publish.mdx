In this tutorial you’ll install Otso and publish your first note.

## Install prerequisites

- Node.js 18+
- pnpm

Then install:

```bash
pnpm install -g otso
```

## Initialize workspace

```bash
otso init mysite
cd mysite
```

## Publish a note

Create `hello.md` and run:

```bash
otso publish hello.md
```

Open the generated site to see your post.
